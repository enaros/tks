Basic workflow

show

The first screen will look like this.
You have to select the target Deployment and Language you want to edit and a Filter. The Filter will narrow the keys to those which starts/contains/ends with the filter.
It is recommended to use always a filter to narrow the search, otherwise you will end up editing 4000+ keys.
Additional options will target different sets of keys.

Once you click the button, you can:
- start editing right away, just type in and press TAB (or SHIFT + TAB to go backwards) to move on the next/previous translation key
- add a Comparator Column, which will help you to compare different deployments/languages
Every change is automatically saved when you move between rows.
If you would like to set a particular key to __EMPTY__, press CTRL + SPACE.

Add a comparator column

Press the button to see the following modal dialog:

Once you add new columns, you will be able to compare between deployment/languages.
Hotkeys will automatically copy the content from the comparator column to the translation key that you are editing. Just press ALT + N where N is the hotkey for that column.
You can start all over pressing the button again.

Stats

show

You will see a column chart for each language in every deployment where your User has READ PERMISSIONS
Each color represents a language, and grey colors represents translation keys that have not been completed yet.
Mouse over a column to see detailed numeric information.

Bulk copies

show

You can copy all the translation keys from a Deployment + Language to a different target Deployment.
You can select between two options:
- REPLACE + CREATE will create new translation keys and replace the keys from target with the ones found in source
- ONLY CREATE will only create those translation keys that have not been found in target Deployment
